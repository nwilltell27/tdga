# TDGA - Terrible Disc Golf Association
Welcome to the Terrible Disc Golf Association. This website allows users to create personal player profiles and connect to a database to "add discs to your bag."

## Technologies
- HTML5
- CSS3
- JavaScript
- MongoDB
- Express
- Mongoose
- Node
